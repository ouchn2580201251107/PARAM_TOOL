### 2.1 参数表管理视图

#### 2.1.1 ParameterTableListView

**文件路径**：`parameter/views.py`

**功能**：参数表清单列表展示，支持筛选和搜索

**方法**：`get(self, request)`

**逻辑流程**：
1. 获取查询参数（domain, status, search）
2. 查询ParameterTable对象
3. 根据条件筛选（domain、status）
4. 根据关键词搜索（name、business_description）
5. 统计业务领域列表
6. 渲染模板

**模板**：`table_list.html`

**日志记录**：
```python
logger.info(f"[ParameterTableListView] 开始查询参数表清单")

```
